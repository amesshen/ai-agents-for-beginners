## 🌱 시작하기

이 강의는 AI 에이전트를 구축하는 기본 개념을 다룹니다. 각 강의는 고유한 주제를 포함하고 있으니 원하는 곳부터 시작하세요!

이 강의는 다국어 지원을 제공합니다. [이곳에서 지원하는 언어를 확인하세요](#-multi-language-support).

생성 AI 모델을 처음 사용해보는 경우, 21개의 레슨으로 구성된 [초보자를 위한 생성 AI](https://aka.ms/genai-beginners) 강의를 확인해보세요.

이 저장소에 [별(🌟)을 눌러](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) 주시고 [포크하여](https://github.com/microsoft/ai-agents-for-beginners/fork) 코드를 실행하는 것을 잊지 마세요.

### 다른 학습자 만나기, 질문 답변 받기

AI 에이전트 구축 관련하여 어려움이 있거나 질문이 있으면, [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord) 내 전용 Discord 채널에 참여하세요.

### 필요한 것들

이 강의의 각 레슨에는 코드 예제가 있으며, code_samples 폴더에서 확인할 수 있습니다. [저장소를 포크하여](https://github.com/microsoft/ai-agents-for-beginners/fork) 본인만의 복사본을 만드세요.

이 실습의 코드 예제들은 Microsoft Agent Framework와 Azure AI Foundry Agent Service V2를 활용합니다:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure 계정 필요

이 강의는 Microsoft의 다음 AI 에이전트 프레임워크 및 서비스를 사용합니다:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

강의 코드를 실행하는 방법에 대한 자세한 내용은 [강의 설정](./00-course-setup/README.md)을 참조하세요.
